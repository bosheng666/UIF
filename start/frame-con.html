压缩包文件结构

在ZIP文件中，你可以找到准备在你的项目中使用的所有的CSS、JavaScript和图片文件。UIF 的核心框架几乎没有任何样式，这样做是为了保持它的苗条。因此我们提供两个额外的风格样式，一个渐变样式和一个扁平化样式。每个风格样式提供单独的CSS文件以及压缩后的版本。

文件夹 概述
/css 包含 UIF 所有的CSS文件和压缩后的版本。
/images 包含 UIF 所有的Images文件和压缩后的版本。
/js 包含 UIF 所有的JavaScript文件和压缩后的版本。 
/css
    <!-- UIF的基本样式 -->
    reset.css
    component.css       //组件公共样式，后面组件以单独样式表存在，后面统一压缩到component.css样式表；

   
 

/js
    <!-- JavaScript和压缩后的版本 -->
   jquery-1.6.4 .min.js
   component.js       //组件公共js库，后面组件以单独js库存在，后面统一压缩到component.js文件中

搭建项目并使用

首先，确保你有一个可靠的代码编辑器，比如 Sublime Text 。然后在你需要的HTML的文档头部引用框架的CSS和JavaScript文件（最好是压缩后的版本,根据功能需求添加）。 就是这样！


ps:别忘了添加 jQuery类库的引用;引用的文件注意上下的加载顺序。

示例

<!DOCTYPE html>
<html lang="zh-CN" >
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
	    <meta name="renderer" content="webkit" />
	    <meta http-equiv="X-UA-Compatible" content="IE=Edge,chrome=1" /> 
        <title>多多框架-hello world!</title>
        <link rel="stylesheet" href="../../static/common/css/reset.css" />
        <link rel="stylesheet" href="../../static/common/css/component.css" />
        <script src="../../static/common/js/jquery-1.6.4 .min.js"></script> 
    </head>
    <body>
    </body>
</html>
