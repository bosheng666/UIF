标准化

减少HTML元素在不同浏览器下默认样式的不一致。

这个组件基于由 Nicolas Gallagher 建立的 Normalize.css 项目。它采用了与传统的CSS重置不同的方法，它只针对需要进行标准化的样式进行标准化，从而避免了冗余的代码。标准化组件总是被优先加载，所以你不必担心你的项目在浏览器中出现冲突。

修改

在 UIF 中，我们对 Normalize.css 做了一些修改和改进。

  • 将 <mark> 和 <h1> 作为默认元素放入 基础组件 中
  • 修改 <fieldset> 默认值为 0
  • 为 radio 和 checkbox 元素添加对应的指针
  • 设置表单控件框大小为 border-box
  • 修改 :disabled 选择器
  • 为 <code>、 <kbd>、 <pre> 和 <samp> 带来更好的 font-family
  • 移除占位符在火狐浏览器中的透明属性
