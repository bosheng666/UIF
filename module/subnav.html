子导航

定义一个不同风格样式的子导航。

用法

若要应用此组件，添加 .uk-subnav 类到一个 <ul> 元素并在列表中嵌套 <a> 元素。使用 <span> 替代 <a> 元素来禁用某个项目。

示例

  • Item
  • Item
  • Disabled
  • Item

Code.

<ul class="uk-subnav">
    <li><a href="">...</a></li>
    <li><a href="">...</a></li>
    <li><span>...</span></li>
    <li><a href="">...</a></li>
</ul>

修饰

子导航分隔线

添加 .uk-subnav-line 类用线条将菜单项单独隔开。

示例

  • Item
  • Item
  • Disabled
  • Item

Code.

<ul class="uk-subnav uk-subnav-line">
    <li>...</li>
</ul>

子导航 pill

添加 .uk-subnav-pill 类来高亮显示背景经过的菜单项。要对菜单项添加被选中的状态，只需添加 .uk-active 类。

示例

  • Active
  • Item
  • Disabled
  • Item

Code.

<ul class="uk-subnav uk-subnav-pill">
    <li class="uk-active">...</li>
</ul>

具有下拉菜单的子导航

这是一个如何使用具有 下拉菜单 的子导航示例。

示例

  • Active
  • Item
  • Disabled
  • More
    • Item
    • Another item
    • Header
    • Item
    • Another item
    • Separated item

Code.

<ul class="uk-subnav">
    <li><a href="">...</a></li>

    <!-- 这是启用JavaScript的容器 -->
    <li data-uk-dropdown="{mode:'click'}">

        <!-- 这是切换下拉菜单的导航项 -->
        <a href="">...</a>

        <!-- 这是下拉菜单 -->
        <div class="uk-dropdown uk-dropdown-small">
            <ul class="uk-nav uk-nav-dropdown">
                <li><a href="">...</a></li>
            </ul>
        </div>
    </li>
</ul>
