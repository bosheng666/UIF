关闭

定义一个可以与不同组件进行组合样式的关闭按钮。

用法

若要应用这个组件，添加 .uk-close 类到一个 <a> 或 <button> 元素。

示例

Code.

<a href="" class="uk-close"></a>

样式修饰

添加 .uk-close-alt 类使用另一种样式来替代默认的关闭按钮。

示例

Code.

<a href="" class="uk-close uk-close-alt"></a>
