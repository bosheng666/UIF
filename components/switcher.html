切换器/Switcher

通过不同的内容窗格进行动态变换。

用法

切换器由数个触发单元和与他们相关联的内容项组成。添加 data-uk-switcher="{connect:'#ID'}" 属性到包含触发单元的元素，并且指向包含内容项的元素的id。添加 .uk-switcher 类到同一个元素。通常，切换器和其他组件搭配使用，这里将展示其中一部分。

Code.

<!-- 这是触发单元的容器 -->
<ul data-uk-switcher="{connect:'#my-id'}">
    <li><a href="">...</a></li>
</ul>

<!-- 这是内容项的容器 -->
<ul id="my-id" class="uk-switcher">
    <li>...</li>
</ul>

注意 效果组件 中的 .uk-margin 类被用在一些例子中，用于在切换器中添加适当的间距。

切换器与按钮

切换器同样可以应用于 按钮或按钮组 中。只需要添加切换器data属性到一个包含了一组按钮的 <div> 元素中或者添加到带有 .button-group 类的元素。

示例

Link
  • Hello!
  • Hello again!
  • Bazinga!
Link
  • Hello!
  • Hello again!
  • Bazinga!

Code.

<!-- 这是用于切换的按钮的容器 -->
<div data-uk-switcher="{connect:'#my-id'}">
    <button class="uk-button" type="button">...</button>
</div>

<!-- 这是包含了内容项的容器 -->
<ul id="my-id" class="uk-switcher uk-margin">...</ul>

<!-- 这是包含了用于切换的按钮的按钮组 buttons -->
<div class="uk-button-group" data-uk-switcher="{connect:'#my-id'}">
    <button class="uk-button" type="button">...</button>
</div>

<!-- 这是包含了内容项的容器  -->
<ul id="my-id" class="uk-switcher uk-margin">...</ul>

JavaScript 事件

每当切换器组件切换到一个内容区域时，都会触发一次 uk.switcher.show 事件。

示例

$('[data-uk-switcher]').on('uk.switcher.show', function(event, area){
    console.log("Switcher switched to ", area);
});
