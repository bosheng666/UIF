下拉菜单

为可切换的下拉菜单定制不同的样式。

用法

任意内容，比如一个按钮，都可以作为切换器用来切换下拉菜单。只需用一个带 data-uk-dropdown 属性的 <div> 元素将其包裹即可。添加 .uk-dropdown 类到一个子级的 <div> 元素来创建下拉菜单本身。一个下拉菜单可以通过鼠标悬停或者的点击的方式来切换。

Data属性 概述
data-uk-dropdown 鼠标悬停打开下拉菜单，并添加一个很小的延迟，这样下拉菜单便不会在你停止在切换器上悬停时立即消失。
data-uk-dropdown="{mode:'click'}" 通过点击切换器来打开下拉菜单。再次点击，下拉菜单便关闭。

重要 为了应用一个下拉菜单，其父元素拥有一个能恰当地将二者对其的相对位置是很重要的。很多组件默认地创建了这样的定位场景，比如 按钮，导航栏，子导航 和 选项卡 等组件。

    <div class="uk-dropdown">...</div>

</div>

    <!-- 这是用来切换下拉菜单的元素 -->
    <div>...</div>

    <!-- 这是下拉菜单 -->
    <div class="uk-dropdown">...</div>

</div>

调整下拉菜单

调整下拉菜单，只需添加 data-uk-dropdown="{justify:'#ID'}"属性。需要调整的下拉菜单的父元素需要有一个标记id，这样下拉菜单便会扩宽度至这个被标记的元素的宽度。

示例

  • Item
  • Another item
  • Header
  • Item
  • Another item
  • Separated item

Code.

<!-- 这是需要调整的下拉菜单的父元素 -->
<div id="my-id">

    <!-- 这是关联了JavaScript的容器 -->
    <div class="uk-button-dropdown" data-uk-dropdown="{justify:'#my-id'}">

        <!-- 这是切换下拉菜单的元素 -->
        <button class="uk-button">...</button>

        <!-- 这是下拉菜单 -->
        <div class="uk-dropdown">...</div>
    </div>
</div>

响应式行为

在狭窄的视口中，比如手机上，可能没有足够的空间来扩展下拉菜单。在这种情况下，下拉菜单会翻转它的对齐准线。如果还是没有足够的空间，网格列会占满宽度并在下拉菜单中垂直地堆叠。

缩小调整

默认情况下，下拉菜单有一个固定的宽度并且文字会切换到下一行。如果你想要你的下拉菜单更小巧，使它不必换行便能扩展到内容的宽度，添加 .uk-dropdown-small 类即可。这对一个按钮下拉菜单是很有用的。

JavaScript 选项

这是关于如何通过属性设置选项的示例：

data-uk-dropdown="{mode:'hover'}"
选项 可选值 默认值 概述
mode hover, click hover 下拉菜单的触发行为
remaintime integer 800 悬停模式下，自动关闭下拉菜单前的等待时间
justify CSS selector false 按指定元素的宽度拉伸下拉菜单
boundary window CSS selector 用被标注的元素来保持下拉菜单的可见性
delay integer 0 悬停模式下，下拉菜单被显示前的延迟时间，以毫秒为单位。

事件

你可以为自定义的功能绑定回调事件。

事件名 概述
uk.dropdown.show 下拉菜单显示时触发
uk.dropdown.stack 当下拉菜单堆叠以适应屏幕时触发

示例

$('[data-uk-dropdown]').on('uk.dropdown.show', function(){
    // custom code...
});
