面包屑

创建一个面包屑导航，显示用户在网站中的当前位置。

用法

面包屑组件由并排排列的链接和链接之间的间隔组成。

类 概述
.uk-breadcrumb 在 <ul> 元素中添加这个类来定义面包屑组件。使用 <a> 元素作为列表中面包屑的项目。
.uk-active 在列表项中添加这个类，对其应用一个选中状态，并且使用 <span> 来替代 <a> 元素。

使用一个 <span> 替代 <a> 元素，来禁用面包屑的某个项目。

示例

  • Home
  • Blog
  • Category
  • Post

Code.

<ul class="uk-breadcrumb">
    <li><a href="">...</a></li>
    <li><a href="">...</a></li>
    <li><span>...</span></li>
    <li class="uk-active"><span>...</span></li>
</ul>
