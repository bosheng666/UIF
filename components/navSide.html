导航菜单

为列表导航菜单定义不同的风格样式。

用法

要应用这个组件，为最外面的 <div> 元素添加 .c-navs 类。 使用 <a> 元素作为列表中的菜单项。要对一个菜单项应用选中状态的效果，添加 .z-cur 类即可。

示例

注意 默认情况下，导航菜单没有任何样式。这就是为什么要添加一些具有样式的修饰类，这是很重要的，在例子中我们使用了 .c-navs 类。

Code.

 
      <div class="c-navs">
          <ul>
              <li>
                  <a href="" class="z-cur">基本侧边栏导航选中</a>
              </li>
              <li>
                  <a href="" >基本侧边栏导航1</a>
              </li>
              <li>
                  <a href="" >基本侧边栏导航2</a>
              </li>
              <li>
                  <a href="" >基本侧边栏导航3</a>
              </li>
          </ul>
      </div>

嵌套式导航菜单

你可以很容易向菜单导航添加任意数量的 <ul> 元素。

类 概述
.c-navs-sub 在第一个嵌套的 <ul> 中添加这个类，使其拥有最佳的间距。
.uk-nav-parent-icon 给导航菜单添加这个类，可以使父菜单项增加图标。

示例

Code.


      <div class="c-navs-sub">
          <ul>
              <li>
                  <a href="" class="z-cur">基本侧边栏导航选中</a>
              </li>
              <li>
                  <a href="" >基本侧边栏导航1</a>
                  <ul>
                      <li>
                        <a href="" class="z-cur">基本侧边栏导航选中</a>
                      </li>
                      <li>
                        <a href="" >基本侧边栏导航1</a>
                      </li>
                      <li>
                        <a href="" >基本侧边栏导航2</a>
                      </li>
                      <li>
                        <a href="" >基本侧边栏导航3</a>
                      </li>
                  </ul>
              </li>
              <li>
                  <a href="" >基本侧边栏导航2</a>
              </li>
              <li>
                  <a href="" >基本侧边栏导航3</a>
              </li>
          </ul>
      </div>
